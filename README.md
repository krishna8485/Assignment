# Assignment


Follow the below steps to execute the project.

1) Follow the instruction given in https://gradle.org/install/ for gradle installation.
2) Change the file path to read airport data from csv file.
3) Go to build.gradle file location and execute 'gradle clean build' commond.
4) After the build is success, execute the 'gradle bootRun' commond in project location.
5) Use 'http://localhost:8080/welcome' to access the application.
6) Use 'http://localhost:8080/swagger-ui.html' to access swagger-ui.
